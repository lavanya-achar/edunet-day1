Understanding Block and Inline Elements

Block-Level Elements

A block-level element starts on a new line and takes up the full width available. Examples include:

  • Paragraphs
  • Headings
  • Divisions

Inline Elements

An inline element does not start on a new line and only takes up as much width as necessary. Examples include:

This text is inside a span element, and this is bold text. Here is a link.

This is a paragraph (block-level) containing inline elements, like this bold text and italicized text. You can also add hyperlinks.
